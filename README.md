# react-native-turntable

一个基于 React Native 实现的转盘 App。

## 主要功能

- 支持创建、编辑、删除多个转盘
- 每个转盘可自定义选项
- 转盘动画旋转，随机选择结果
- 结果本地持久化，重启不丢失
- 支持安卓/iOS

## 安装与运行

```bash
npm install
npm start
```

如需在安卓或 iOS 设备运行：

```bash
npm run android
npm run ios
```

## 技术栈

- React Native (Expo)
- Redux Toolkit + redux-persist
- react-native-svg（转盘绘制）
- react-native-reanimated（动画）
